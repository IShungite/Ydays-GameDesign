# Nuriki

## GDD

### 1.Instant summary

#### a.Overview

- Nuriki est un MMORPG reprenant les bases de Metin2, un jeu se déroulant dans un univers médiévial fantastique chinois.

#### b.Look & feel

- C'est un mmo simplicistes graphiquement, avec des contrôles de skill abordables. Une fois rentré de votre travail, vous n'aurez qu'une envie c'est de taper des mobs et de vous défouler sur ce jeu.

#### c. Plot and setting

- 3 empires se font la guerre depuis fort longtemps, vous incarnerez une des 4 races disponibles, et avancerez dans cette aventure remplie de mystères. Vous aurez la possibilité de rejoindre ou de créer une guilde, pour ensemble, détruire vos énnemies.

### 2. Gameplay

#### a. User interface (menu nav)

- Lors du lancement du jeu, vous attérisserez sur l'écran de connexion. Une fois connecté, vous pourrez choisir votre peresonnage, ou en créer un. Une fois IG, vous aurez plusieurs fenêtres à votre disposition, toutes utilent dans leur domaine.

#### b. Developement (3c pour vous)

- Troisième personne, caméra 360°, 3D. Marche, cours, chevauche, ramasse des objets. Déplacement ZQSD, espace pour taper, W pour ramasser, I pour ouvir l'inventaire
